# Тестовое задание

Выполнил - Круглов Леонид

## Установка зависимостей

Для сборки необходим `bun`. Процесс установки можно найти по ссылке [https://bun.com/docs/installation#macos-%26-linux]

Для установки всех необходимых библиотек:

```shell
bun i
```

Проект использует ПоискКино API ([https://poiskkino.dev/]). Для отправки запросов используется токен, который необходимо получить через телеграмм-бота и указать в файле `.env`. В файле `.env.default` показан пример указания токена.

## Запуск

Для запуска необходимо прописать:

```shell
bun run dev
```

Проект будет запущен на порте [http://localhost:5173]
